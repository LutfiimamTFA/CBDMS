# **App Name**: WorkWise

## Core Features:

- Task Management: Create, assign, and manage tasks with detailed descriptions, statuses, assignees, dates, and priorities.
- Time Tracking: Track time spent on tasks automatically upon Clock In/Out, providing real-time updates on work duration.
- Work Reporting: Generate comprehensive work reports based on tracked time and task status. These reports provide insight into employee output and project progression.
- Dependency Management: Define relationships between tasks (dependencies and subtasks) to ensure organized workflow.
- Smart Task Suggestion: Use AI to analyze past work patterns and suggest tasks and estimate duration of future tasks. This tool predicts what the user might need to work on next.

## Style Guidelines:

- Primary color: Vivid blue (#4285F4) for a clean, trustworthy feel.
- Background color: Light blue (#E3F2FD), a very desaturated shade of the primary hue, provides a calm and professional backdrop.
- Accent color: Green (#34A853) to highlight key actions and completed tasks.
- Font pairing: 'Poppins' (sans-serif) for headings and 'PT Sans' (sans-serif) for body text to ensure readability and a contemporary feel.
- Modern and minimalist icons to represent task statuses, priorities, and actions.
- Clean, intuitive layout inspired by ClickUp with a focus on drag-and-drop functionality and customizable dashboards.
- Subtle transitions and animations to enhance user engagement and provide visual feedback on interactions.
